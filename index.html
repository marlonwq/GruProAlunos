Bem-vindo ao GruPro Alunos!

Repositório para alunos dos cursos de extensão do GruPro (Grupo de Programação da UFBA).

Compartilhamento de soluções, códigos e conexões.

Projeto Não-Oficial

Este é um repositório NÃO OFICIAL, criado e mantido de forma independente por um aluno que participou do curso. Ele não possui vínculo formal com a GruPro ou com a UFBA.

O que você encontra aqui?

  • Acesso a diferentes soluções para os problemas apresentados durante o curso.
  • Um espaço para expressar sua maneira de pensar, promovendo conexão entre os estudantes.
  • Um tutorial de contribuição para projetos de código aberto no GitHub.

Uso Básico e Ético

Sinta-se à vontade para compartilhar suas soluções e descobrir novas ideias!

Este repositório é um espaço de aprendizado colaborativo. Use-o para comparar abordagens e descobrir novas formas de pensar, sempre de forma ética.

Aviso Importante

Recomendo não utilizar este repositório como "gabarito" ou copiar soluções. O curso é para aprender e a equipe de instrutores está à sua disposição.
